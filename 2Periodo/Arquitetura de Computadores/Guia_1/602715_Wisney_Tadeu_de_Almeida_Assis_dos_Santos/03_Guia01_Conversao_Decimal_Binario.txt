03.) Fazer as conversões de decimal para a base indicada:
a.) 74(10) = X(4) = 1022(4)
	74 % 4 = 2
	18 % 4 = 2
	04 % 4 = 0
	01 % 4 = 1

b.) 47(10) = X(8) = 57(8)
	47 % 8 = 7
	05 % 8 = 5

c.) 64(10) = X(16) = 40(16)
	64 % 16 = 0
	04 % 16 = 4
	
d.) 184(10) = X(16) = B8(16)
	184 % 16 = 8
	011 % 16 = B(11)

e.) 124(10) = X(16) = 7C
	124 % 16 = C(12)
	007 % 16 = 7
